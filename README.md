## Портативный D&D Мастер для ВК!

## Используюмые технологии:
- Python 3.12.2
- Ollama 0.17.1rce (минимум)
- SQLite
- Библиотека vkbottle для работы с API

#### 1) Создать сообщество во ВКонтакте и получить API ключ сообщества. 
> Подробнее по ссылке: https://dev.vk.com/ru/api/access-token/community-token
#### 2) Открыть папку с проектом и вставить ключ доступа ВК в файл окружения *.env*.
> VK_TOKEN=<ваш_токен>
#### 3) Запустить *setup.bat* - он установит все необходимое, 
включая приложение для локальной работы с LLM = Ollama
#### 4) Затем запустить *start.bat*. 
Обратите внимание, что в трее у вас запуститься Ollama, когда завершите работу с ботом, не забудьте выключить его.
#### 5) Откроется 2 окна:
- *Brain DnD* - отвечает за нейросеть
- *Bot DnD* - отвечает за отладку бота
#### 6) Теперь, перейдя в созданное сообщество, **создайте чат**
#### 7) Введите `!commands` - вам откроется список возможных команд
#### 8) Введите `!session new <имя_сессии>`
#### 9) Поздравляю! Можете начинать играть, отправляя сообщения боту
